Termial. HW_2
1. Сделать папку dir_1
Ввести команду mkdir dir_1 и нажать Enter

 2. Зайти в папку dir_1
 Ввести команду cd dir_1 и нажать Enter
 
 3. Создать папку inner_dir_1
 Ввести команду mkdir inner_dir_1 и нажать Enter
 
 4. Посмотреть где ты находишься
 Ввести команду pwd и нажать Enter
 
 5. Находясь в папке dir_1 создать пустой текстовый файл tf_1.txt
 Ввести команду touch tf_1.txt и нажать Enter
 
 6. Находясь в папке dir_1 через команду cat создать текстовый файл tf_2.txt со следующими строками:
- the first 1
- the second 2
- the third 3
Ввести команду cat > tf_2.txt ввести указанные строки и нажать Enter

 7. Зайти в папку inner_dir_1
 Ввести команду cd inner_dir_1 и нажать Enter
 
 8. Через cat сделать текстовый файл tf_3.txt  c любыми строками
 Ввести команду cat > tf_3.txt ввести любые строки и нажать Enter
 
 9. Через cat добавить в текстовый файл tf_3.txt строку “the second 2”
 Ввести команду cat >> tf_3.txt ввести “the second 2” и нажать Enter
 
 10. Через cat добавить в текстовый файл tf_3.txt строку “the sec 2”
 Ввести команду cat >> tf_3.txt ввести “the sec 2” и нажать Enter
 
 11. Через cat добавить в текстовый файл tf_2.txt строку “the sec 3”
 Ввести команду cat >> ../tf_2.txt ввести “the sec 3” и нажать Enter
 
 12. Через cat добавить в текстовы й файл tf_3.txt строку “the SeCoNd 2”
 Ввести команду cat >> tf_3.txt ввести “the SeCoNd 2” и нажать Enter 
 
 13. Через cat добавить в текстовый файл tf_2.txt строку “the seConD 2”
 Ввести команду ccat >> ../tf_2.txt ввести “the seConD 2” и нажать Enter
